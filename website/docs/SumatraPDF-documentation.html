SumatraPDF is a free PDF, EPUB, MOBI, CHM, XPS, DjVu, CBZ and CBR reader for Windows.
It's small and starts up fast.
Simplicity of user interface is important to us.

Downloads:

  • latest stable release
  • pre-release builds (latest features, potentially less stable)
Found a bug? Have feature request? Submit via issue tracker.
Have feedback or question? Discuss in forums.
Want to make SumatraPDF work better in your language? Contribute a translation.
