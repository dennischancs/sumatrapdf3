How to submit bug reports
Did you find a bug in Sumatra? Please create a bug report in our issue tracker.
Does it involve a specific PDF (or CHM or XPS or ...) file? Please attach it to the issue (try dragging and dropping on the issue field). Without a test document we rarely can make progress on fixing the issue.
If you're using the latest release version, please try the latest daily build. It's possible that bug has already been fixed.
If the file (or other information) is confidential, you can e-mail it directly to kkowalczyk@gmail.com (Krzysztof Kowalczyk, Sumatra's main developer).
If you report a crash, please:
  • press Cancel when Sumatra shows message box about a crash
  • this should launch a Notepad with the content of the crash log
  • post the content of the crash log as a gist (https://gist.github.com/). For example, Ctrl-A to select all text, Ctrl-C to copy it to a clipboard and then Ctrl-V to the gist
  • link the gist to bug report at https://github.com/sumatrapdfreader/sumatrapdf/issues
